# لوحة الاتصال (LiteDialer)

تطبيق اتصال أندرويد حقيقي، مصمم خصيصاً ليكون خفيف جداً على الأجهزة الضعيفة:

- بدون AppCompat / Material Design Components / RecyclerView / Compose
- بدون أي صور PNG — كل الأيقونات vector XML خفيفة
- بدون أي تخزين أو مكتبات شبكة
- حجم APK النهائي متوقع أقل من 1 ميجابايت
- يدعم أندرويد 5.0 (API 21) فما فوق — يغطي تقريباً كل الأجهزة القديمة والضعيفة

## كيف تبنيه (مجاني بالكامل)

1. حمّل **Android Studio** مجاناً من: https://developer.android.com/studio
2. افتح البرنامج → اختر **Open** → اختر مجلد `LiteDialer` هذا كامل
3. انتظر لين يخلص الـ Gradle Sync أول مرة (بيحتاج إنترنت المرة الأولى بس عشان يحمّل أدوات البناء)
4. من القائمة العلوية: **Build → Build Bundle(s) / APK(s) → Build APK(s)**
5. لما يخلص البناء، رح تطلعلك رسالة فيها رابط **locate** — دوس عليه وبيوديك على مكان ملف الـ APK
6. انسخ ملف الـ APK ع الموبايل وثبته (لازم تفعّل "السماح بالتثبيت من مصادر غير معروفة" بإعدادات الموبايل)

## ملاحظة عن التوقيع

الـ APK يلي رح يطلع من "Build APK(s)" هوي **debug build** — شغال 100% على جهازك وتقدر تنزله فوراً وتجربه مجاناً.
إذا بدك تنشره على متجر Google Play لاحقاً، بتحتاج تعمل **Generate Signed Bundle/APK** بدل هيك، وهاد كمان مجاني بالكامل من نفس البرنامج.

## بنية المشروع

```
LiteDialer/
├── app/
│   ├── src/main/
│   │   ├── java/com/lite/dialer/MainActivity.kt   ← كل منطق التطبيق بملف واحد
│   │   ├── res/layout/activity_main.xml            ← الواجهة كاملة
│   │   ├── res/values/                              ← الألوان والنصوص والأنماط
│   │   └── res/drawable/                            ← أيقونات vector خفيفة
│   └── build.gradle
└── build.gradle
```
